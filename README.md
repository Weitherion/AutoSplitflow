
# Auto Split Flow
基于 Solana 链上的自动分帐器，让资金分配透明、可视、自动执行。

特点：实时链上资金流3D可视化。

## 主要功能
1. 立即自动分帐。
   > 用户配置好收款人和分帐比例，点击执行，即可立即分帐。
2. 自动生成分帐收款地址。
   > 用户只需配置收款人和分账比例，即可生成一个“智能收款地址”，所有进入该地址的资金都会自动按比例分账到设定的地址中。实现每一笔收入都自动分帐。
3. NFT链上记账。
   > 每笔分帐都会生成一个NFT保存在发送账户中，便于统计分析账目。


## 交互流程
1. 填入收款参与地址（多个 Solana 地址）；
2. 每个地址的分账比例（如 70 / 20 / 10）；
3. 点击“执行”或者“生成自动分帐地址”；

* 执行：立即完成分帐，并生成NFT收据（NFT不可转移）。
* 生成自动分帐地址：系统返回一个 Solana 地址； 任何人向该地址转账 USDC / SOL，合约都将自动按比例分账，钱直接进入各自钱包，并生成NFT收据（NFT不可转移）。


## 资金流3D可视化
视觉冲击点：资金在空间中分裂并流动。

资金流 = 粒子流动
每个钱包 = 一个球体
转账 = 一条线
金额 = 粒子数量
方向 = 动画流动
