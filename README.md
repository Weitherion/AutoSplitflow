
# Auto Split Flow
基于 Solana 链上的自动分帐器。

主要功能：
1. 自动生成分帐收款地址。
   > 用户只需配置收款人和分账比例，即可生成一个“智能收款地址”，所有进入该地址的资金都会自动按比例分账到设定的地址中。实现每一笔收入都自动分帐。

2. NFT链上记账。
   > 每笔分帐都会生成一个NFT保存在发送账户中，便于统计分析账目。


## 自动生成分帐收款地址
1. 填入收款参与地址（多个 Solana 地址）；
2. 每个地址的分账比例（如 70 / 20 / 10）；
3. 点击「生成收款地址」；
4. 系统返回一个 Solana 地址； 任何人向该地址转账 USDC / SOL，合约都将自动按比例分账，钱直接进入各自钱包。
5. 生成一个NFT，记录分帐详情，绑定在发送地址中，且不可转移。
